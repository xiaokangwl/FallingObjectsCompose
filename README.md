# FallingObjectsCompose
点击某处从点击处，弹出一个自定义图片或其它的物体，点击后弹出然后自由落体到底部，最后渐变消失


### 效果预期

1. **点击屏幕**
   - 每次点击都会在点击位置生成一个图片。

2. **弹出动画**
   - 图片会从点击点向上弹出 150 像素。

3. **自由落体**
   - 图片从弹出的最高点自由落体到底部，始终保持垂直方向。

4. **旋转效果**
   - 图片会随机旋转，但旋转不会引起位置偏移。

5. **渐变消失**
   - 图片在底部停留 1 秒后，逐渐变透明并完全消失。
